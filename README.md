# Android-SpeechStructureApp

## Overview

SpeechStructureApp is a native Android application developed for an Upwork client to support structured speech and writing through a rich-text editing experience.
The app focuses on clean UI flows, editor integration, and local persistence.


## Features & Tech Stack
- Rich-text editing using HTML-based editor inside WebView
- JavaScript â†” Android bridge for editor interaction
- Multi-screen onboarding flow with ViewPager2
- BottomSheet dialogs for user actions
- Local storage with SharedPreferences + Gson
- Built with Java, Android Fragments, and Material Components
